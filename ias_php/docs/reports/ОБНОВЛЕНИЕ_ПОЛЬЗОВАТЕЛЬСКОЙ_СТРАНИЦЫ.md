# Обновление пользовательской страницы "Мои заявки"

## Выполненные изменения

Внесены изменения в пользовательскую страницу "Мои заявки" согласно требованиям:

### 1. **Перемещение кнопки "Создать заявку"**
- **Было**: Кнопка располагалась в правой части заголовка (col-md-4)
- **Стало**: Кнопка перемещена в дальний правый конец заголовка (col-md-6)
- Заголовок "Мои заявки" теперь занимает левую половину (col-md-6)
- Кнопка "Создать заявку" занимает правую половину (col-md-6)

### 2. **Обновление шрифта в колонках "Вложения" и "Формат вложений"**

#### Колонка "Вложения":
- **Шрифт**: Увеличен с 10px до 14px
- **Цвет**: Изменен с серого (text-muted) на черный (#000)
- **Стили кнопок**: Обновлены для лучшей читаемости
- **Отступы**: Увеличены для лучшего отображения

#### Колонка "Формат вложений":
- **Шрифт**: Увеличен с 10px до 14px  
- **Цвет**: Изменен с серого (text-muted) на черный (#000)
- **Стиль**: Убран badge-стиль, заменен на обычный текст
- **Вес шрифта**: Установлен как normal для лучшей читаемости

### 3. **Обновленные CSS стили**
```css
.attachments-list .btn {
    padding: 2px 6px;        /* Увеличены отступы */
    font-size: 14px;         /* Увеличен шрифт */
    line-height: 1.4;        /* Улучшена высота строки */
    color: #000 !important;   /* Черный цвет */
}
```

#### Заголовок:
- ✅ Кнопка "Создать заявку" перемещена в дальний правый конец
- ✅ Заголовок "Мои заявки" занимает левую половину
- ✅ Сохранен зеленый фон заголовка

#### Колонки таблицы:
- ✅ **"Вложения"**: Шрифт 14px, цвет черный
- ✅ **"Формат вложений"**: Шрифт 14px, цвет черный

- ✅ Сохранена функциональность скачивания файлов

### 5. **Сохраненная функциональность**
- ✅ Скачивание вложений по клику
- ✅ Отображение иконок файлов
- ✅ Подсказки с именем и размером файла
- ✅ Фильтрация и сортировка таблицы
- ✅ Пагинация

Изменения улучшают читаемость и соответствуют требованиям пользователя по расположению элементов и стилизации текста.
