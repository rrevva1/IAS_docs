# ИАС УТС — информационно-аналитическая система учета технических средств

Репозиторий проекта ИАС учета технических средств предприятия (Help Desk, учет активов, справочники, аудит).

- **Приложение:** [ias_uch_vnii/](ias_uch_vnii/) — веб-приложение на PHP (Yii2), PostgreSQL.
- **Docker:** [docker/README.md](docker/README.md) — запуск приложения и БД на любой ОС (вся конфигурация в папке `docker/`).
- **Документация:** [docs/README.md](docs/README.md) — навигатор по всей документации.
- **Для ВКР:** [docs/vkr/README.md](docs/vkr/README.md) — какие документы использовать по разделам выпускной работы.

## Два контура документации

1. **Документация на ИАС** — описание системы, требования, модель данных, регламенты, планирование (см. [docs/README.md](docs/README.md)).
2. **Материалы для ВКР** — те же документы плюс методика оформления ТЗ и навигатор по разделам ВКР (см. [docs/vkr/README.md](docs/vkr/README.md)).
