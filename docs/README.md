# Документация проекта ИАС УТС

Краткий навигатор по документации репозитория. Два основных контура использования:

1. **Документация на ИАС** — описание системы, требования, модель данных, регламенты, планирование.
2. **Материалы для ВКР** — те же документы плюс методика оформления ТЗ и навигатор по разделам ВКР: [docs/vkr/README.md](vkr/README.md).

---

## Дубли и связи

- **sistema/ТРЕБОВАНИЯ_К_СИСТЕМЕ.md** — выдержка раздела 5 из **TZ.md**; при изменении требований править TZ.md, при необходимости обновлять выдержку.
- **Доработки.md** (корень) и **planning/ДОРАБОТКИ_СТАТУС.md** — связаны: первый перечень доработок, второй — таблица статуса.
- **bd/ОПИСАНИЕ_И_ПЛАН_ДЕЙСТВИЙ_БД** и **bd/МОДЕЛЬ_БД** — разные: план изменений и рекомендации vs описание текущей/целевой модели.
- Документы по парсингу ОУ (import_ou/): **регламент** — единый источник правил; ETL и «первые 10 строк» ссылаются на регламент.
