# Документация для написания ВКР

Навигатор по материалам проекта, полезным для выпускной квалификационной работы (ВКР).

## Планы и шаблоны ВКР (в этой папке)

| Документ | Описание |
|----------|----------|
| [ПланВКР.txt](ПланВКР.txt) | План работы: тема, введение, цель, описание предметной области, модель хранения, входные/выходные данные, план по главам (1–3). |
| [План_ВКР(примерный).txt](План_ВКР(примерный).txt) | Примерная структура ВКР: исследовательский, проектный, технологический разделы; введение, цель, план работы. |
| [План_вкр_Рязанцев(пример).txt](План_вкр_Рязанцев(пример).txt) | Пример плана ВКР (ИАС учёта технической документации) — образец структуры и объёма. |
| [План_ВКР_ИАС_учета_ТС.txt](План_ВКР_ИАС_учета_ТС.txt) | План ВКР по теме ИАС учёта технических средств предприятия (по образцу Рязанцева, не менее того же объёма). |

---

## Два контура использования

1. **Документация на ИАС** — описание системы, требования, архитектура, модель данных (для разделов ВКР «Объект и предмет», «Требования», «Проектирование»).
2. **Методика и реализация** — правила формирования ТЗ, план реализации, отчёты о разработке (для разделов «Методы», «Реализация», «Внедрение»).
