# Рабочие документы системы ИАС учёта технических средств

В этой папке приведён перечень **рабочих документов**, которые требуются подобной информационно-аналитической системе для проектирования, разработки и эксплуатации. Для каждого документа указано: **что это за документ**, **назначение** и **расположение** в репозитории (исходный файл — единый источник правды; при необходимости сюда можно добавить копии или выдержки).

Документы приведены в порядок и актуализированы по состоянию проекта; при изменении содержания править следует исходные файлы в корне проекта и в `docs/`.

## Обновление документов

- Изменения в требованиях вносятся в **TZ.md**; при необходимости обновляется выдержка в sistema/ТРЕБОВАНИЯ_К_СИСТЕМЕ.md.
- Изменения в модели данных — в **bd/МОДЕЛЬ_БД_ias_vniic.md** и скриптах схемы БД.
- Изменения в правах и регламенте безопасности — в **sistema/РОЛИ_И_ПРАВА_ДОСТУПА.md** и **sistema/РЕГЛАМЕНТ_БЕЗОПАСНОСТИ.md**.
- Полный навигатор по всей документации: [../README.md](../README.md). Перечень документации и привязка к ВКР: [../vkr/ПЕРЕЧЕНЬ_ДОКУМЕНТАЦИИ.md](../vkr/ПЕРЕЧЕНЬ_ДОКУМЕНТАЦИИ.md).
